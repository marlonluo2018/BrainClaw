# Start Personal Assistant

## Quick Start

Say to AI: **"hi"** or **"hello"**

AI will read brain files and confirm ready.

## Full Startup

```
Read all files in ./assistant_brain/
- SOUL.md
- CONFIG.md  
- skills/*/
- memory/*.md
- logs/current.md
```

## What Happens

1. AI reads your preferences and history
2. Checks yesterday's uncompleted tasks
3. Suggests email check if overdue
4. Ready for your requests
